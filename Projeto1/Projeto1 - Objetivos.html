Sumário

O objetivo desta pesquisa é melhorar o processo de recuperação de dados geográficos oferecidos por uma infraestrutura de dados espaciais. Para isto, o projeto tem como objetivo o desenvolvimento de um sistema que vai facilitar a localização e o acesso a dados geográficos oferecidos pela SDI. A pesquisa tem ainda os seguintes objetivos específicos:

  • o desenvolvimento de uma nova abordagem para melhorar a localização de dados geográficos, com base apenas em restrições espaciais;
  • o desenvolvimento de uma nova abordagem para melhorar a localização de dados geográficos, com base apenas em restrições temáticas;
  • o desenvolvimento de uma nova abordagem para melhorar a localização de dados geográficos, com base apenas em restrições temporais;
  • o desenvolvimento de uma nova abordagem para melhorar a localização de dados geográficos, com base em mais de uma dimensão;
  • o desenvolvimento de uma nova abordagem para melhorar a localização de serviços de dados geográficos, com base em mais de uma dimensão;
  • o desenvolvimento de uma nova abordagem para realizar a anotação temática automática de dados geográficos;
  • o desenvolvimento de um portal geográfico, que permita aos clientes do sistema utilizarem todas as abordagens desenvolvidas de forma conveniente; e
  • a aplicação a solução desenvolvida à infraestrutura de dados espaciais brasileira, que será usada como o principal estudo de caso ao longo de todo o projeto.
