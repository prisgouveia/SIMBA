Sumário

O WMS e o WFS permitem que um cliente acesse as várias camadas (feature types) oferecidas em um conjunto de dados utilizando uma interface aberta e padronizada. Entretanto, quando provedores de dados geográficos cadastram o seu conjunto de dados no serviço de catálogo da SDI, eles normalmente utilizam um único registro de metadados, que descreve o conjunto de dados como um todo. Esta característica acaba impondo uma série de limitações para a resolução de consultas que envolvem qualquer uma das três dimensões inerentes a dados geográficos: espaço, tempo e tema.

Com relação à dimensão espacial, a principal limitação dos catálogos atuais ocorre porque normalmente uma única região geográfica é definida para representar todas as camadas oferecidas pelo serviço. Isto faz com que o catálogo considere que todas as camadas oferecidas pelo serviço cobrem exatamente a mesma região geográfica, o que normalmente não acontece. Tal característica prejudica a cobertura e a precisão das consultas com este tipo de restrição, o que limita a qualidade dos resultados obtidos durante este processo. Ademais, nenhum ranking efetivo é oferecido para este tipo de consulta. Desta forma, serviços que cobrem exatamente a mesma região requisitada em uma consulta são considerados tão relevantes quanto os serviços que cobrem apenas uma parte da mesma. Os mesmos problemas inerentes à dimensão espacial podem ser estendidos para a dimensão temporal. Entretanto, na dimensão temporal, tais problemas são agravados pelo fato de que a extensão temporal do conjunto de dados é muitas vezes omitida pelo provedor no momento do registro.

Algumas limitações importantes também ocorrem durante a resolução de consultas temáticas. Atualmente, os catálogos usam palavras-chaves para descrever os temas relacionados a um conjunto de dados geográficos. Assim, os relacionamentos semânticos existentes entre os termos usados nas consultas e os termos usados na descrição dos dados, tais como sinonímia, meronímia/holonímia, hiperonímia/hiponímia não podem ser identificados, reduzindo a cobertura e precisão dos resultados. Certamente, o uso dos conceitos desenvolvidos na web semântica [BERNERS-LEE et al., 2001], aliado ao uso de mecanismos para descrever formalmente a semântica dos serviços oferecidos pela SDI, irão corroborar para a minimização desta problemática. Por fim, um ranking dos resultados baseado na semântica também não existe nos serviços de catálogos oferecidos pelas SDI atuais, o que, novamente, reduz a qualidade dos resultados apresentados.

Recentemente, muitos trabalhos de pesquisa têm sido propostos para resolver a recuperação de dados geográficos em infraestruturas de dados espaciais e portais geográficos. Os trabalhos desenvolvidos variam de acordo com a abordagem usada para resolver o problema e o tipo de recurso que pode ser recuperado. Por exemplo, algumas soluções propostas [LEMMENS et al., 2007] [LUTZ, 2007] [YUE et al., 2011] [BAI et al., 2009] [STOCK et al., 2010] [LI et al., 2011] permitem a recuperação de serviços, mas não permitem a recuperação de informação em nível de camadas. Outros trabalhos desenvolvidos [JANOWICZ et al., 2008] [ZHANG et al., 2010] [LIU et al., 2011] [ZHANG et al., 2013] permitem a recuperação de camadas, mas não consideram as informações dos serviços durante este processo. Algumas pesquisas desenvolvidas [LUTZ & KOLAS, 2007] [SMITS & FRIIS-CHRISTENSEN, 2007] [ATHANASIS et al., 2009] [KALABOKIDIS et al., 2014], por sua vez, não focam na recuperação de nenhum tipo de recurso específico, e não propõem qualquer medida de ranking para organizar os resultados que serão apresentados para o cliente.

Destarte, apesar dos avanços obtidos com o desenvolvimento dos serviços de catálogo atuais, há uma clara necessidade de se melhorar o processo de recuperação dos dados geográficos oferecidos por uma SDI. A melhoria deste processo, além de prover um maior acesso aos dados existentes, vai facilitar o reuso destes dados para a produção de novas informações, bem como melhorar a qualidade de diversos processos de tomada de decisão nos quais dados geográficos desempenham um papel fundamental, tais como planejamento urbano, gestão ambiental, prevenção e gestão de desastres e o gerenciamento de recursos naturais. Todas estas contribuições ajudam a destacar a importância e a relevância deste projeto.
